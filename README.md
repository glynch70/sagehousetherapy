# Sage House Therapy

A production-ready Next.js foundation for Sage House Therapy.

## Tech stack

- Next.js App Router
- TypeScript
- Tailwind CSS
- ESLint
- Prettier
- Vercel-ready project structure

## Development

```bash
npm install
npm run dev
```

## Quality checks

```bash
npm run lint
npm run format:check
npm run build
```
